# INF2011S – Systems Design & Implementation  
## Project 1: Phumla Kamnandi Hotels – Guest Booking System

**Course:** INF2011S  
**Department:** Information Systems  
**Year:** 2025  

---

## Project Overview

This project implements a **C#.Net-based Guest Booking System** for the Phumla Kamnandi Hotels group. The aim is to complete **one iteration of the systems development life cycle**, covering design, implementation, testing, and demonstration.

The system supports hotel receptionists in managing guest bookings through a relational database–driven application.

---

## Scope

- Make a telephone booking  
- Change an existing booking  
- Cancel a booking  
- Enquire on guest bookings  
- Occupancy level report (date-based)  
- One additional value-adding report  


---

## Deliverables

- Systems Specification Document  
- Working C#.Net Application  
- Relational Database  
- Test Plan and Test Cases  

---

## Technologies Used

- C#.Net  
- Relational Database (SQL)  
- Object-Oriented Design  
- UML (Sequence, Class, ER Diagrams)
